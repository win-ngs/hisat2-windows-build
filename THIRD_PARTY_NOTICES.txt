Packaged runtime files
----------------------

The release ZIP includes HISAT2 executables, PowerShell wrapper scripts,
runtime DLLs, and a bundled bzip2 command-line tool copied from an
MSYS2-UCRT64 installation. Package ownership was
checked with `pacman -Qo`, and package metadata was checked with `pacman -Qi`.

HISAT2 executables and wrappers
  Built from HISAT2 2.2.2 source with the Windows/MSYS2-UCRT64 compatibility
  patch in this repository.

bzip2 / libbzip2
----------------

The release ZIP bundles the bzip2 command-line tool (`bzip2.exe`) and its
library (`libbz2-1.dll`) so that the PowerShell wrapper can read and write
`.bz2` reads on Windows without requiring a separate bzip2 installation.
gzip (`.gz`) is handled in-process by the wrapper via the .NET runtime and
needs no bundled tool.

Upstream project:
  https://sourceware.org/bzip2/

MSYS2 package:
  mingw-w64-ucrt-x86_64-bzip2 1.0.8-3
