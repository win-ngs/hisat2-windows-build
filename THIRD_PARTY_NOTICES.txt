Packaged runtime files
----------------------

The release ZIP includes HISAT2 executables, PowerShell wrapper scripts, and
runtime DLLs copied from an MSYS2-UCRT64 installation. Package ownership was
checked with `pacman -Qo`, and package metadata was checked with `pacman -Qi`.

HISAT2 executables and wrappers
  Built from HISAT2 2.2.2 source with the Windows/MSYS2-UCRT64 compatibility
  patch in this repository.
